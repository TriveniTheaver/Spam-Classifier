# Spam Classifier

This project classifies SMS messages as spam or not spam using Machine Learning.

## Project Overview
The dataset contains labeled SMS messages. The goal is to predict whether a message is spam (1) or ham (0) using Naive Bayes.

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Jupyter Notebook

## Model
Naive Bayes Classifier

## Evaluation
- Accuracy
- Confusion Matrix
- Classification Report

## Output
The model predicts whether a message is spam and visualizes the results.
